**版本**：v1.0  
**软件名称**：根图分析 - RootVision  
**开发者**：根系分析团队

## 1. 概述

### 1.1 软件简介

RootVision（根图分析）是一款专业的植物根系图像分析软件，运行于 Windows 10/11 平台。软件对根系扫描图像或照片进行自动化处理，依次完成图像增强、OTSU 阈值分割、形态学过滤、连通组件计数和形态参数计算，并支持批量并行处理。

### 1.2 适用场景

- 平板/光学扫描仪获取的根系图像自动化分析
- 相机拍摄的根系照片分析（需搭配比例尺参照物）
- 根长、根面积、根体积、根表面积、平均直径、比根长等参数计算
- 按直径分级的根系分类与统计
- 多张图像的高通量批量处理

### 1.3 术语

| 术语 | 定义 |
|------|------|
| OTSU（大津法） | 自动确定最优二值化阈值，最大化前景/背景类间方差 |
| 连通组件 | 二值图像中相互连接的前景像素集合，每个组件代表一段独立根系 |
| 骨架化 | 将根系区域细化为单像素宽度的中心线，用于根长测量 |
| 比例尺 | 图像像素到物理毫米的转换系数（mm/pixel） |
| ROI | 感兴趣区域（Region of Interest），手动框选的分析范围 |
| 比根长 | 总根长与总根面积的比值（mm/mm²），反映资源获取效率 |
| 直径（DIA） | 通过矩形模型估算：直径 = 面积 / 长度 |

## 5. 操作指南

### 5.1 启动

双击 `RootVision.exe` 启动软件。启动后显示主窗口，状态栏显示「就绪」。

### 5.2 第一步：加载图像

点击「选择图像」按钮或 Ctrl+O，选择根系图像文件（支持 jpg/jpeg/png/bmp/tiff/tif）。

加载成功后，图像显示在预览画布中，状态栏显示图像像素尺寸。可使用滚轮缩放、拖拽平移查看细节。

### 5.3 第二步：校准比例尺

比例尺决定了像素与物理尺寸的转换关系，是**所有计算结果的前提**。软件提供 **4 种设置方式 + 1 种参照物选取**：

**方式① 通过测量（已知长度换算）**
1. 在比例尺面板输入已知实际长度（mm），默认 10.0
2. 输入该长度在图像中对应的像素数，默认 100
3. 点击「设置比例」→ 弹出确认对话框，显示换算结果

**方式② 通过图像尺寸**
1. 输入图像的实际宽度和高度（mm），默认均为 100.0
2. 点击「设置尺寸」→ 根据图像像素尺寸自动计算

**方式③ 通过 DPI**
1. 输入扫描分辨率 DPI，默认 300
2. 点击「DPI 转换」→ 按公式 `25.4 / DPI` 自动换算

**方式④ 手动输入**
在「过滤参数」面板直接修改「像素转换系数（mm/像素）」的数值。

**方式⑤ 参照物选取（推荐，最直观）**
1. 点击「选取参照物」按钮，光标变为十字准星
2. 在图像上沿已知物理长度的参照物拖拽画一条线（起点绿色圆点，终点红色圆点）
3. 松开鼠标后弹出对话框，输入该参照物的实际长度（mm）
4. 自动计算 mm/pixel 并显示确认信息

### 5.4 第三步（可选）：调整分析参数

**直径分级**（面板「直径分级设置」）

默认 5 级标准：

| 名称 | 最小直径 | 最大直径 |
|------|----------|----------|
| 超细根 | 0.0 mm | 0.5 mm |
| 细根 | 0.5 mm | 1.0 mm |
| 中根 | 1.0 mm | 2.0 mm |
| 粗根 | 2.0 mm | 5.0 mm |
| 超粗根 | 5.0 mm | ∞（无上限） |

- **添加**：弹出对话框填写名称、最小直径、最大直径（留空 = 无上限）
- **编辑**：在 Treeview 表格中选中一行后修改
- **删除**：删除选中级别（至少保留 2 级，否则弹出警告）
- **恢复默认**：确认后重置为上述 5 级标准

**过滤参数**（面板「过滤参数」）

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 像素转换系数 | DoubleVar | 0.1 | mm/pixel，即比例尺 |
| 分割方法 | 标签 | 大津法（固定） | OTSU 自动阈值 |
| 最小根系大小 | IntVar | 20 | 连通组件计数时过滤小于此像素数的组件 |
| 最小根长 | IntVar | 5 | 过滤长度低于此值（像素）的区域 |
| 最小根面积 | IntVar | 20 | 过滤面积低于此值（像素）的区域 |
| 最小纵横比 | DoubleVar | 1.0 | 过滤长宽比低于此值的非根团块 |

**扫描优化**（菜单「设置 → 扫描图像优化」）

对于扫描仪图像，启用后可获得更好的分割效果。系统自动根据背景类型和质量调整增强策略。

### 5.5 第四步（可选）：ROI 框选

如果只需分析图像局部区域：

1. 点击「选取分析区域」按钮（按钮文字变为「取消框选」）
2. 在图像上拖拽矩形框（蓝色虚线，半透明填充）
3. 分析时将仅处理 ROI 内的像素

点击「清除 ROI」恢复全图分析。ROI 选区在切换图像后仍然有效，需手动清除。

### 5.6 第五步：选择输出格式

在「输出设置」面板中勾选需要的结果文件：

| 选项 | 变量 | 默认 | 生成的文件 |
|------|------|------|-----------|
| 保存 CSV 数据 | save_csv_var | ☑ | `根系分析结果.csv` |
| 保存 JSON 数据 | save_json_var | ☑ | `根系分析结果.json` |
| 保存结果叠加图 | save_overlay_var | ☑ | `{图像名}_结果叠加_{时间戳}.png` |
| 保存详细报告图 | save_report_var | ☐ | `{图像名}_详细报告_{时间戳}.png` |
| 在图中显示比例尺 | show_scale_var | ☑ | 控制叠加图和报告图中是否绘制比例尺 |

### 5.7 第六步：执行分析

点击「开始分析」（或 Ctrl+A）。

分析在线程中异步执行，不阻塞界面。进度条和状态栏实时更新。

**分析流程（7 步）**：

```
1. 加载图像           → progress 10%
2. 图像增强（+扫描优化）→ progress 30%
3. OTSU 二值化分割     → progress 30%
4. 形态学过滤          → progress 50%
5. 连通组件计数        → progress 60%
6. 参数计算+直径分级    → progress 70%
7. 保存结果文件        → progress 90%
完成                   → progress 100%，弹出完成提示
```

分析完成后：
- 结果文本自动显示在右侧
- 可在「视图」菜单切换查看各阶段图像
- 点击「显示详情」查看直径分布弹窗
- 点击「复制结果」将结果文本复制到剪贴板

### 5.8 查看结果

右侧结果文本区包含：

**图像信息**：路径、尺寸、像素比例  
**基本参数**：根系数量、总根长（mm）、根系面积（mm²）、总根体积（mm³）、总根表面积（mm²）、平均直径（mm）、比根长（mm/mm²）  
**直径分级统计**（每个级别）：数量、体积、体积占比（%）、平均直径  
**过滤统计**：总区域数、过滤区域数、移除区域数、移除百分比

弹窗「显示详情」和「显示过滤统计」提供更详细的格式化输出。

### 5.9 批量处理

1. **添加文件**：点击「批量选择」（Ctrl+Shift+O），选择多张图像。文件名显示在批量列表 Listbox 中
2. **设置参数**：确认比例尺、过滤参数、输出选项（批量共用当前设置）
3. **设置超时**：超时时间默认 660 秒，单张图像超时将自动跳过
4. **选择模式**：
   - 「批量分析」：顺序逐张处理
   - 「并行批量分析」（Ctrl+P）：最多 5 个进程并发处理，大幅提升速度
5. **等待完成**：进度条和状态更新

批量完成后生成：
- 每张图像的独立结果文件（CSV/JSON/叠加图/报告图）
- `batch_summary_report_{时间戳}.txt`（批量汇总报告）
- `batch_statistics_chart_{时间戳}.png`（批量统计图表）

### 5.10 退出

点击「退出程序」/ 文件 → 退出 / Ctrl+Q。

退出时自动保存以下配置到 `config.json`，下次启动自动恢复：
- 像素转换系数（比例尺）
- 过滤参数（最小根长/面积/纵横比/根系大小）
- 直径分级标准（含自定义级别）
- 输出选项（CSV/JSON/叠加图/报告图勾选状态）
- 输出目录路径

## 6. 输出文件

### CSV 文件（`根系分析结果.csv`）

中文表头，包含字段：
- 图像名称、图像路径、图像尺寸、像素比例
- 根系数量、总根长(mm)、根系面积(mm²)、总根体积(mm³)、总根表面积(mm²)、平均直径(mm)、比根长(mm/mm²)
- 各级别名称_数量、各级别名称_体积(mm³)、各级别名称_体积占比(%)、各级别名称_平均直径(mm)
- 总区域数、过滤区域数、移除区域数、移除百分比

### JSON 文件（`根系分析结果.json`）

与 CSV 相同内容的结构化 JSON，便于程序化处理。

### 结果叠加图（`{图像名}_结果叠加_{时间戳}.png`）

原始图像上以绿色透明层标记检测到的根系区域。可选显示比例尺标注。

### 详细报告图（`{图像名}_详细报告_{时间戳}.png`）

18×12 英寸的多面板综合报告（matplotlib figure），包含 7 个子图：
1. 原始图像 + 尺寸信息
2. 增强/扫描优化图像
3. 分割结果 + 区域数
4. 过滤结果
5. 根系计数彩色标记图
6. 检测结果叠加图 + 参数摘要
7. 直径分布柱状图（各级别体积占比）

## 7. 注意事项

1. **比例尺是核心**：直接影响所有长度/面积/体积结果。强烈建议每次分析前使用参照物方式校准
2. **图像质量**：高对比度 + 均匀光照 = 最佳效果。扫描图像建议白色背景 + 300 DPI
3. **背景色类型**：默认浅色背景（根系深、背景浅）。若分析荧光/深色背景图像，务必在设置中切换背景类型为 `dark`，否则分割结果会反转
4. **过滤参数调优**：先用默认值分析 → 切换到过滤图查看效果 → 根据实际微调参数。参数过大误删根系，过小保留噪点
5. **直径分级**：各级别范围必须连续且不重叠。最后一个级别建议最大直径留空（代表 ≥ 最小值）
6. **批量内存**：并行模式同时加载 5 张图像，建议 8GB 以上内存。超大尺寸图像（＞5000×5000 像素）建议使用串行模式
7. **ROI 持久**：ROI 选区跨图像保留。分析新图像前若不需要 ROI，请手动点击「清除 ROI」
8. **结果保存**：分析结果不会自动保存，请及时导出。程序关闭仅保存配置参数
9. **输出目录**：默认路径为桌面「根系分析结果」文件夹。首次使用建议确认或修改
